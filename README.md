# demo-node-express
Demo Node.js and Express.js HTTP server

## installation

````
npm i
````

## execution

````
node index.js
````

## references

Node.js
[https://nodejs.org/en/](https://nodejs.org/en/)

Express.js
[https://expressjs.com/](https://expressjs.com/)

## benchmark

Install autocannon

````
npm i -g autocannon
````
